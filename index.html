Welcome to my Ordering Visualizer!
This is a short explanation on how this works:
Select an specific sorting algorithm by selecting the "Algorithms" menu.
Adjuts the speed of the simulation with the slider.
You will be given a random array with random length and values. If you want a new one, click the "New Array" button.
Press start to initialize the simulation. A new simulation cannot be started until you finish the last one.
